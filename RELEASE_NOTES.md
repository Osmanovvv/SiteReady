# SiteReady v1.0

Аудит сайта «под ключ»: один прогон обходит сайт и выдаёт единый отчёт по
**SEO · тех/QA · производительности · доступности · адаптивности** — с общим баллом,
оценкой A–F, приоритизированными проблемами и советами «как починить». Запуск одной
командой: `node server.js`.

## Что внутри
- **Движок** — чистый Node.js, **ноль внешних зависимостей** (только встроенные модули):
  BFS-обход, прощающий HTML-парсер, SSRF/egress-защита, модель скоринга.
- **Фронт** — React + Vite SPA (TanStack Router, Tailwind, shadcn/ui), собранный в статику
  и отдаваемый движком; связь по SSE-контракту (`meta · progress · done · error`).
- **Отчёт** — циферблат балла, 5 категорий с бейджами достоверности, список проблем с
  фильтром и советами, постраничная таблица, клиентский отчёт, экспорт в PDF и автономный HTML.

## Надёжность
- SSRF-защита (IPv4/IPv6, metadata, mapped/NAT64), декомпрессия gzip/deflate/raw/brotli,
  редиректы/таймауты/лимит распаковки и байт-бюджет обхода, кодировки utf-8/windows-1251.
- SPA-сайты получают честный баннер вместо десятков ложных critical; внешние 403/405/429/таймаут
  не помечаются «битыми»; недостижимый домен даёт быстрое понятное сообщение.
- Тесты: **78 движок + 5 фронт** (герметичные, без сети наружу). Чеклист приёмки §15 — 21/21.

## Запуск
```bash
git clone https://github.com/Osmanovvv/SiteReady.git
cd SiteReady
node server.js            # → http://127.0.0.1:3000
```
`web/dist` в репозитории — клон запускается без сборки. Для разработки фронта:
`cd web && npm install && npm run build`.
